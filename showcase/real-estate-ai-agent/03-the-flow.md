# The Full Flow

## System Architecture

### Step 01 — Inquiry Arrives

LINE OA message or web widget. Agent responds in <5 seconds, 24/7. First reply: detect language (Thai / EN / RU), acknowledge, open qualification.

### Step 02 — Qualification Loop

5 structured questions delivered conversationally. Budget → Location → Property type → Bedrooms → Timeline. If user drops off mid-flow, re-engage once after 2 hours.

### Step 03 — Lead Scoring

System scores the lead A/B/C based on budget range, timeline urgency, and response completeness. Score determines routing priority.

### Step 04 — Property Matching

Agent queries live property sheet. Returns max 3 matching units with key specs. Offers photos (link) or viewing booking.

### Step 05 — Booking & Handoff

Slot selection → calendar block → agent notification via LINE Notify. Human agent gets: name, contact, score, matched property, budget confirmed, timeline.

### Step 06 — Post-Viewing Follow-up

T+24hrs: automated check-in message. T+72hrs: if no response from human agent, flag in lead log. Keeps pipeline visible.
